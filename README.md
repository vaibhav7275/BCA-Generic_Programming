# BCA Generic Programming Programs

Is repository mein mere BCA ke C aur C++ programs hain.

## Topics
- Generic Programming (Function & Class Templates)
- Linked List (Creation)
- User-defined functions (e.g., multiply)
- Other basic C/C++ programs

## How to run
- C programs:
  - `gcc filename.c -o output`
  - `./output`
- C++ programs:
  - `g++ filename.cpp -o output`
  - `./output`
